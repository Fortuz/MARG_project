UR software jöhet ide
- "doc" mappa dokumentáció
- "src" mappa source kód

URconnect.py - Alapszintű socket kezelő a robothoz
RTDEhandler.py - RTDE kezelő az adatok kiolvasásához, részletek a ur honlapján. (30004-es port)
robotprog.txt - Szinkronizált robotirányításhoz mintaprogram (URScript nyelven, 30002-es porton kell feltölteni), nektek majd a movej és servoj, illetve movel parancsok kellenek.
scheduler.py - Egy példa alkalmazás a bonyolultabb esetre, nektek valószínűleg nem fog ilyen kelleni.
(A kódok formázatlanok, tisztítsátok őket nyugodtan, ha kérdés van jelezzétek!)
